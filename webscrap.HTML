Web scraping Yelp Reviews

  • Home

Since everyone loves food (presumably), the ultimate end goal of this Project will be to acquire the data to answer some questions and hypotheses about the restaurant scene in Pittsburgh

We will Scrap the restaurants data in Pittsburgh from the Yelp API and with this data, We retrieve the comments/reviews and ratings from users on restaurants.

Learnings

Used Beautiful Soup library for getting data out of HTML, XML

Used requests library to make Http requests to specified URI using GET, POST, PUT, HEAD requests

Authenticated HTTP Request with the Yelp API

Results

Aquired all of the restaurants in Pittsburgh (on Yelp)

Retrieved all of the Business objects. Paginated by querying 20 restaurants each request

Using BeautifulSoup, parsed the HTML of a single Yelp restaurant page to extract the reviews in a structured form
